# Blog_Management_System

In this project, I developed a backend Web API application using ASP.NET Core Web API. It includes complete CRUD operations for managing multiple users, posts, and comments, along with functionality to count total comments by each individual user and upload files. The application is structured following the N-tier architecture, consisting of Controller,BLL, and DAL for database operations. It also follows SOLID principles to ensure clean, maintainable, and scalable code.
